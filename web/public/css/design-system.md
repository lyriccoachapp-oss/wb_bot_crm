# 🎨 WorkBangers CRM — Дизайн-система

## Описание

Визуальный стиль: **светлая, чистая, профессиональная** admin-панель.
Вдохновление: Materialize, DashLite, Herozi, Admindek.

## 📐 Layout

### Sidebar (боковая панель)
- Ширина: **260px**
- Фон: `#2b2c40` (тёмный)
- Положение: `fixed`, слева
- Навигация: иконки SVG + текст
- Секции: заголовки uppercase (0.6875rem)
- Активный пункт: `background: #7367f0`, белый текст, shadow

### Topbar (верхняя панель)
- Высота: **64px**
- Фон: белый + тень
- Sticky (прилипает к верху при скролле)
- Слева: заголовок страницы
- Справа: бейдж роли, аватар пользователя

### Content area
- Padding: **28px**
- Карточки: белые, `border-radius: 10px`, `box-shadow`

## 🧩 Компоненты

### Кнопки

| Класс | Вид |
|-------|-----|
| `.btn.btn-primary` | Фиолетовый фон, белый текст |
| `.btn.btn-secondary` | Прозрачный, border |
| `.btn.btn-success` | Зелёный фон |
| `.btn.btn-danger` | Красный фон |
| `.btn.btn-sm` | Уменьшенный размер |
| `.btn-auth` | Полноширинный для auth страниц |

### Бейджи

| Класс | Цвет |
|-------|------|
| `.badge--primary` | Фиолетовый |
| `.badge--success` | Зелёный |
| `.badge--warning` | Оранжевый |
| `.badge--danger` | Красный |
| `.badge--info` | Голубой |
| `.badge--secondary` | Серый |

### Таблицы
- Заголовки: uppercase, серый, 12px, border-bottom 2px
- Строки: hover → `#f8f8fb`
- Ячейки: padding 14px 16px

### Модальные окна
- Overlay: чёрный 50% + blur
- Карточка: белая, max-width 600px, анимация slideIn
- Закрытие: `.modal.active` toggle

### Формы
- Поля: белый фон, border `#ebe9f1`, radius 8px
- Focus: border primary + shadow 3px primary-light
- Сетка: `.row-grid` — 2 колонки, collapsed на мобильных

### Пагинация
- Контейнер: `.pagination`
- Элементы: 36x36px, border-radius 8px
- Активный: primary фон, белый текст

## 📱 Breakpoints (адаптивность)

| Брейкпоинт | Назначение |
|------------|------------|
| `≤ 1024px` | Sidebar скрывается, бургер-меню |
| `≤ 768px` | Мобильный layout, 1 колонка |
| `≤ 640px` | Формы в 1 колонку |
| `≤ 480px` | Уменьшенные отступы |

## ⚠️ Правила

1. Все комментарии в CSS — **на русском языке**
2. Отступы — **табуляцией** (4 символа)
3. Без сторонних UI-библиотек
4. Нативные HTML, CSS, JavaScript
5. Шрифт только **Rubik** (Google Fonts)
