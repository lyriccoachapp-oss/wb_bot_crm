# Документация Telegram Bot

Приложение развернуто в изолированном Docker-контейнере как составная часть CRM-системы.

## Структура проекта
* `Dockerfile` / `docker-compose.yml` - Развертывание контейнера.
* `index.php` - Основной WebHook Endpoint бота для Telegram.
* `lib/` - Библиотеки логики (взаимодействие с Laravel API, i18n, Telegram).
* `lang/` - Словари переводов (ru.json, uk.json, en.json).
* `webapp/` - Веб-приложения (Учет времени: `wt.php`, Сканер чеков: `receipts.php`), открываемые внутри интерфейса Telegram WebApp.

## Запуск
Для сборки и запуска приложения внутри изолированного окружения выполните:

```bash
docker compose up -d --build
```
*Контейнер запустится и начнет прослушивать локальный порт `8071`.*

## Архитектура
* Все обращения от Nginx, адресованные в `crm.workbangers.com/bot/*`, проксируются внутренним редиректом напрямую в Docker-контейнер `127.0.0.1:8071`.
* Бот не имеет прямого доступа к базе данных `MySQL`. Вместо этого он обращается к внутреннему Laravel API (`X-Bot-Token`).
* Поддерживается языковая функция `__()` для динамического перевода текстов интерфейса.
